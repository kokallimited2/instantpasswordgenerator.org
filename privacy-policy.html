Privacy Policy

How we handle your data — Instant Password Generator

Updated: 7 May 2026 · Controller: Kokal Operations Ltd, UK · ICO registered

Summary: No passwords are collected. All generation is client-side using crypto.getRandomValues(). Personal data is only collected if you contact us.

1. Data Controller

Instant Password Generator is operated by Kokal Operations Ltd, registered in England and Wales. Privacy enquiries: privacy@instantpasswordgenerator.org

2. What We Collect

What we do NOT collect

  • Generated passwords — all generation uses crypto.getRandomValues() client-side. No password is sent to any server.
  • Keystroke data, clipboard contents, or typed input of any kind.

Collected automatically

GitHub Pages logs standard HTTP data: IP address, browser type, pages visited, timestamps. Not used to identify individuals. Retained 30 days.

Voluntarily provided

  • Contact form: name, email, message
  • Cookie preference stored as ipg-ck in browser localStorage — never transmitted to any server
