## ✍️ Adding Blog Posts

Edit `post.json`. Add a new object to `posts[]`:

```json
{
  "id": "7",
  "slug": "your-post-slug",
  "title": "Post Title",
  "date": "7 May 2026",
  "dateiso": "2026-05-07",
  "category": "IT Admin Guide",
  "emoji": "🖥️",
```
