PowerShell Scripts for Secure Bulk Password Generation

3 May 2026·7 min·Alex Chen, IT Security Specialist

Five production-ready scripts for the complete bulk password lifecycle, all using RNGCryptoServiceProvider.

Core Function: CSPRNG Generator

function New-CryptoPassword {
    param([int]$Length=16, [bool]$Symbols=$true, [bool]$NoAmb=$false)
    $pool = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ' + 'abcdefghijklmnopqrstuvwxyz' + '0123456789'
    if ($Symbols) { $pool += '!@#$%^&*()-_=+' }
    if ($NoAmb)   { '0','O','l','1','I' | ForEach-Object { $pool = $pool.Replace($_, '') } }
    $rng = [System.Security.Cryptography.RNGCryptoServiceProvider]::new()
    $b = [byte[]]::new($Length); $rng.GetBytes($b)
    return -join ($b | ForEach-Object { $pool[$_ % $pool.Length] })
}

Bulk CSV Export with Audit Log

$count = 50; $length = 16
$out = "C:\Temp\passwords_$(Get-Date -f 'yyyyMMdd_HHmmss').csv"
$log = "C:\Logs\pwd_gen.log"
1..$count | ForEach-Object {
    [PSCustomObject]@{Index=$_; Password=New-CryptoPassword -Length $length}
} | Export-Csv $out -NoTypeInformation
"$(Get-Date -f 'o') | Generated $count passwords | $env:USERNAME" | Add-Content $log

Bulk Reset Existing Accounts

Get-Content 'C:\Temp\accounts-to-reset.txt' | ForEach-Object {
    $pwd = New-CryptoPassword -Length 16
    $sec = ConvertTo-SecureString $pwd -AsPlainText -Force
    Set-ADAccountPassword -Identity $_ -NewPassword $sec -Reset
    Set-ADUser -Identity $_ -ChangePasswordAtLogon $true
    Write-Host "Reset: $_"
}

Verify Mandatory-Change Flag

Get-ADUser -Filter {WhenCreated -ge (Get-Date).AddDays(-1)} -Properties pwdLastSet |
    Select-Object SamAccountName, @{n='MustChange';e={$_.pwdLastSet -eq 0}} |
    Format-Table -AutoSize
Best practice: Run the verification script the morning after bulk provisioning. Accounts where MustChange is False require manual helpdesk follow-up.
